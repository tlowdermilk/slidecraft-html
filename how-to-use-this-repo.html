How to use this repo.

In this guide

Three steps from cloning the skills folder to a finished page — with or without a mood board. Built for VS Code + GitHub Copilot Chat.

This repo is a tiny style engine. Drop the skills into your own project, point Copilot at your inspiration, and it builds the page for you.

3 steps · ~10 minutes · No build tools required

01 The Steps

From empty repo to finished page.

The whole flow is three moves. Copy the skills folder. Open Copilot Chat. Tell it what you want — with pictures, or just words.

01

Copy the skills folder into your repo.

Grab the .github/skills/ folder from this repo and drop it into the .github/ folder of your own repo in VS Code. That's the whole install. The skills tell Copilot how to design pages in this editorial zine style.

tip · if your repo doesn't have a .github folder yet, just create one at the root.

02

Open GitHub Copilot Chat in VS Code.

Open the chat panel (the speech-bubble icon in the activity bar, or Ctrl+Alt+I). Make sure you're in Agent mode — that's what lets Copilot read the skills folder and write files for you.

tip · the skills load only when your request matches them. You don't have to mention them by name.

03

Describe the page you want — with or without a mood board.

If you have a mood board, drag the images straight into the chat window and tell Copilot what to build. If you don't, just type what you want. The skills carry a default font and theme, so you'll get a polished page either way.

Build a one-page zine-style intro to my product called "Hearth." Use the mood board I just pasted for color and vibe.

tip · be specific about the kind of page (intro, field guide, changelog, landing). The skills branch on that.

02 Two Ways In

With a mood board, or without one.

Both paths work. The mood board gives Copilot a specific aesthetic to chase. The default gives you the warm editorial look this whole repo is built on.

03 What to Say

Sample prompts to copy & paste.

If you're staring at the chat box wondering how to start, here are three that work. Swap in your own topic.

Create a 3-part zine series called "Hearth for new users." Part 01 is the definition, Part 02 is how it works, Part 03 is how to write your first one. Use this repo's default theme.
Make a single landing page for my side project. Here are five mood board images — match the color and energy. Headline copy is "Small tools, sharp edges."
Add a "What's new" page to this series in the same style as the existing parts. Pull the accent color from Part 02.
You're not picking a template. You're giving the agent just enough taste — and the skills do the rest. — Field note · MakeItPretty
Ready · Open Copilot Chat

You have the recipe. Now go build a page.
